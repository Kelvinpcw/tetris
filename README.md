# Tetris Game

A Java implementation of the classic Tetris game.

## Build & Run

```bash
javac *.java
java TetrisGame
```

## Project Structure

- **TetrisGame.java** - Main game class and entry point
- **Board.java** - Game board logic and state management
- **Shape.java** - Tetris piece representations
- **PieceFactory.java** - Factory for creating Tetris pieces

## Current Features

- Basic game board
- Tetris pieces (shapes)
- Game loop
